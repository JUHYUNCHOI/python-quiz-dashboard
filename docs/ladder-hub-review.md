## B. 검토 결과 (2026-06-23)

### 🔴 High — 우선 처리
- **H1 [UX/IA] 필터 과밀** — 토픽 24칩 + 4 row ≈ 36칩. 첫 진입에 칩 벽(모바일 5~6줄) → 선택 마비. → **기본 접기** + 토픽 "인기6+더보기" 또는 4그룹 아코디언.
- **H2 [접근성] 키보드·SR 사용 불가** — ① 검색 input `outline:none`인데 대체 focus 없음(키보드 길 잃음) ② ✓/★/칩/난이도배지에 `aria-label`·`aria-pressed` 없음(SR이 상태·역할 못 읽음) ③ 9.5~10px 흰글자 배지(노랑/주황 배경)·`#94a3b8` 라벨 **대비 4.5:1 미달**. → aria 라벨+pressed, focus-visible outline, 대비 상향.
- **H3 [난이도] 고급→도전 절벽 + 이론토픽 입구 없음** — 도전(63+) 47개(3.3%)뿐. shortestpath 최소 47·unionfind 53·dp 40·tree 40 → 토픽 필터 시 첫 문제가 40~50대 점프. → **띠 경계 62→58**(도전 47→94) 또는 56~66대 충원 + 각 이론토픽 25~35대 entry 1~2개.
- **H4 [UX] "이 수준 더" 어포던스 0** — 난이도 배지 클릭=필터인데 옆 배지(span)와 똑같이 생김 → 발견 불가. → 클릭 단서(▾/배경hover) 추가.
- **H5 [IA] 게스트 인증 불일치** — 허브·algo는 게스트 OK, `/practice` 딥링크는 로그인 벽. 같은 사다리 칸 일부만 열림. → 허브 RequireAuth로 통일 또는 인라인 로그인 안내.
- **H6 [IA] 솔버 2종 분열** — 문제 열면 algo UI vs practice ProblemDetail로 갈림(뒤로가기·헤더 제각각). *알려진 부채.* → 최소 공유 컴포넌트(뒤로·헤더·"막히면 수업")로 행동 통일.

### 🟡 Med
- **M1 "오늘 할 것" 비적응형** — 전역 최저=level 8 "Solve Me First"(외부·자동채점X)를 진도0 학생 1번 추천. → `getAdaptiveNext` 연동 + 외부 후순위.
- **M2 행 배지 과밀(5~6개)** → 모바일 제목 잘림. 우측 ≤3개로.
- **M3 노랑 충돌** — ⭐추천/★즐겨/⏳미룸 다 노랑계열(⏳가 추천처럼 보임). ⏳→중립/잠금색.
- **M4 색 ~39종** + 토픽색↔출처색 겹침(보라·파랑). → 토픽색은 동그라미 테두리만, 행 배지는 무채색.
- **M5 입문 과밀(13~18=273) 평지** + trie L14·graph L13 이상치(토픽평균 대비 40+ 차이) 재평가.
- **M6 journey "지금" 중첩**(수업CTA+허브카드+오늘할것) / 네비 라벨(연습·알고리즘·실전)이 "배우기 vs 풀기" 축 아님.
- **M7 접근성 보강** — input 라벨, 터치타깃 26/20px→32~44, `<h1>`/`<main>`/`<ul>` 시맨틱, 리다이렉트 `aria-live`.
- **M8 "오늘 할 것" 필터종속** 표시 없음(어떤 필터의 다음인지).

### 🟢 Low
keep-all 누락(메모리 반복지적) · 앱톤 불일치(인라인 vs Tailwind/orange-600) · 잔존 라벨(`/practice` breadcrumb "연습(수업별 클러스터)") · 카운트 2곳 중복 · `ladder-last` 솔버경유 미기록 · 거대 단일컴포넌트/매렌더 `LADDER.filter` useMemo화 · 이모지 `aria-hidden`/외부링크 라벨 · 새탭 고지 · KL 세트 39~42·24~27 점프.

### ✅ 배포 안전성
코드 검토 전수검증: 딥링크 1430개 정합(매핑실패 0), 솔버 키 1:1 일치, 리다이렉트 가드 안전(`?cluster=` 가이드·세션 안 깨짐), hook 순서 OK, `lesson_progress`(Supabase) 무관, 보호 localStorage 키 변경 0. **배포 차단급 회귀 없음.** (단 algo 딥링크 url '없음' 지적은 오류 — 300개 전부 deep-link 확인.)
