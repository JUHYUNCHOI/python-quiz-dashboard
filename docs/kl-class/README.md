# KL 사다리 풀이집 (선생님용)

`/course/kl` 의 **40문제 사다리** 전체 풀이. 수업 중 옆에 띄워놓고 답·풀이를 보면서 가르치는 용도.
대상 언어: **C++**. 각 문제마다 핵심 아이디어 + 정답 코드(한국어 주석) + 가르칠 포인트.

> ⚠️ 이 문제들은 외부 공개 저지(LeetCode·CF·CSES·USACO·HackerRank) 문제라 앱 안에서 자동 채점/답 표시가 안 됩니다. 그래서 이 풀이집이 그 역할을 합니다. (앱 자체 문제의 "선생님 모드 답 표시"와는 별개)

## 단계별
- [1단계 · 손풀기 (1~10번)](./ladder-stage1.md) — 전부 쉬움. cin/cout·배열·해시 기본기.
- [2단계 · 기초 도구 (11~21번)](./ladder-stage2.md) — 빈도수·누적합·투포인터 입문.
- [3단계 · 한 번 생각 (22~33번)](./ladder-stage3.md) — 정렬 그리디·sweep·비트마스크 완전탐색·multiset.
- [4단계 · 기법 입문 (34~43번)](./ladder-stage4.md) — 슬라이딩 윈도우·이분탐색(판정)·누적합 mod·그리디.

## 수업 흐름 (커리큘럼 내 위치)
```
문법 레슨(cpp-1~16) ─ 각 레슨 끝마다 on-ramp 3문제
   → [2.연습 단계] 🎯 KL 사다리 40문제 (/course/kl)   ← 이 풀이집이 여기
   → [3.알고리즘] → [4.실전]
```
학생은 `/course/kl` "맵(차례대로)" 탭에서 위→아래 1번씩. 선생님은 이 풀이집으로 따라가며 설명.
